移動スクリプト — PlayerMovement.cs

CODE済
WASD移動 + ジャンプ + 重力処理を実装します。New Input System のポーリング方式に対応。

保存先: Assets/Scripts/Player/PlayerMovement.cs

public class PlayerMovement : MonoBehaviour
{
    // フィールド定義
    public float moveSpeed = 5f;
    public float jumpForce = 5f;
    public float gravity = -9.81f;
    private Vector3 velocity;
    private CharacterController controller;

    void Start() {
        controller = GetComponent<CharacterController>();
    }

    void Update() {
        // WASD入力
        float x = Input.GetAxis("Horizontal");
        float z = Input.GetAxis("Vertical");
        Vector3 move = transform.right * x + transform.forward * z;

        // 接地判定 & ジャンプ
        if (controller.isGrounded && velocity.y < 0)
            velocity.y = -2f;
        if (Input.GetButtonDown("Jump") && controller.isGrounded)
            velocity.y = jumpForce;

        // 重力適用 & 移動
        velocity.y += gravity * Time.deltaTime;
        controller.Move((move * moveSpeed + velocity) * Time.deltaTime);
    }
}
🔧 User操作: Player オブジェクトに PlayerMovement.cs をアタッチ
2-3

視点操作スクリプト — MouseLook.cs

CODE済
マウスによるFPS視点操作。垂直方向は-90°～90°にクランプします。

保存先: Assets/Scripts/Player/MouseLook.cs

public class MouseLook : MonoBehaviour
{
    public float mouseSensitivity = 100f;
    public Transform playerBody;
    private float xRotation = 0f;

    void Start() {
        Cursor.lockState = CursorLockMode.Locked;
    }

    void Update() {
        float mouseX = Input.GetAxis("Mouse X") * mouseSensitivity * Time.deltaTime;
        float mouseY = Input.GetAxis("Mouse Y") * mouseSensitivity * Time.deltaTime;

        xRotation -= mouseY;
        xRotation = Mathf.Clamp(xRotation, -90f, 90f);

        transform.localRotation = Quaternion.Euler(xRotation, 0f, 0f);
        playerBody.Rotate(Vector3.up * mouseX);
    }
}
🔧 User操作: Main Camera に MouseLook.cs をアタッチし、Inspector で PlayerBody に Player をセット

武器・射撃システム

3-1

弾の作成 — Bullet.cs

CODEUSER済
Projectile（物理弾）方式。一定時間で消滅し、衝突時にダメージを与えます。

保存先: Assets/Scripts/Weapon/Bullet.cs

public class Bullet : MonoBehaviour
{
    public float speed = 40f;
    public float lifetime = 3f;
    public int damage = 10;

    void Start() {
        GetComponent<Rigidbody>().velocity = transform.forward * speed;
        Destroy(gameObject, lifetime);
    }

    void OnTriggerEnter(Collider other) {
        Health health = other.GetComponent<Health>();
        if (health != null) health.TakeDamage(damage);
        Destroy(gameObject);
    }
}
🔧 User操作:
① Sphere を作成 →「Bullet」と命名
② Rigidbody をアタッチ（Use Gravity = false）
③ Collider の Is Trigger = true に設定
④ Bullet.cs をアタッチ
⑤ Assets/Prefabs/ にドラッグしてプレハブ化 → シーンから削除

銃スクリプト — Gun.cs

CODE済
弾数管理、リロード、マズルフラッシュ演出を含む汎用銃スクリプト。プレイヤーと敵の両方で使用します。

保存先: Assets/Scripts/Weapon/Gun.cs

public class Gun : MonoBehaviour
{
    public GameObject bulletPrefab;
    public Transform firePoint;
    public ParticleSystem muzzleFlash;  // マズルフラッシュ
    public int maxAmmo = 30;
    public int currentAmmo;
    public float reloadTime = 3f;
    public bool isReloading = false;

    void Start() { currentAmmo = maxAmmo; }

    public void Shoot() {
        if (isReloading || currentAmmo <= 0) return;
        Instantiate(bulletPrefab, firePoint.position, firePoint.rotation);
        currentAmmo--;
        if (muzzleFlash != null) muzzleFlash.Play();
    }

    public IEnumerator Reload() {
        if (isReloading || currentAmmo == maxAmmo) yield break;
        isReloading = true;
        yield return new WaitForSeconds(reloadTime);
        currentAmmo = maxAmmo;
        isReloading = false;
    }
}

ヘルス・ダメージシステム

4-1

体力管理スクリプト — Health.cs

CODE済
PlayerとEnemy両方に使える共通の体力管理スクリプト。死亡時の処理はTagで分岐します。

保存先: Assets/Scripts/Common/Health.cs

public class Health : MonoBehaviour
{
    public int maxHealth = 100;
    public int currentHealth;

    void Start() { currentHealth = maxHealth; }

    public void TakeDamage(int amount) {
        currentHealth -= amount;
        if (currentHealth <= 0) Die();
    }

    void Die() {
        if (gameObject.tag == "Player")
            GameManager.Instance.GameOver();
        else
            Destroy(gameObject);
    }
}
🔧 User操作:
① Player の Tag を「Player」に設定
② Player と Enemy の両方に Health.cs をアタッチ

5-2

敵AIスクリプト — EnemyAI.cs

CODE済
NavMeshAgent でプレイヤーを追跡。攻撃範囲内で自動攻撃＋自動リロード。

保存先: Assets/Scripts/Enemy/EnemyAI.cs

[RequireComponent(typeof(Gun))]
public class EnemyAI : MonoBehaviour
{
    private NavMeshAgent agent;
    private Transform player;
    private Gun gun;
    public float attackRange = 10f;
    public float attackInterval = 1f;
    private float attackTimer = 0f;

    void Start() {
        agent = GetComponent<NavMeshAgent>();
        gun = GetComponent<Gun>();
        player = GameObject.FindGameObjectWithTag("Player").transform;
    }

    void Update() {
        agent.SetDestination(player.position);
        float dist = Vector3.Distance(transform.position, player.position);

        if (dist <= attackRange) {
            attackTimer += Time.deltaTime;
            if (attackTimer >= attackInterval) {
                if (gun.currentAmmo > 0)
                    gun.Shoot();
                else
                    StartCoroutine(gun.Reload());
                attackTimer = 0f;
            }
        }
    }
}
🔧 User操作:
① Enemy の子に「FirePoint」を作成（プレイヤー方向を向くように配置）
② Enemy に Gun.cs と EnemyAI.cs をアタッチ → Inspector で各フィールド設定
③ 完成した Enemy を Assets/Prefabs/ にドラッグしてプレハブ化
④ シーンに複数体の敵を配置

ゲーム管理・UI

6-1

GameManager — GameManager.cs

CODE済
シングルトンパターンで実装。ゲームオーバー時に時間停止・カーソル解放・リスタート機能を提供。

保存先: Assets/Scripts/GameManager/GameManager.cs

public class GameManager : MonoBehaviour
{
    public static GameManager Instance;
    public GameObject gameOverPanel;

    void Awake() { Instance = this; }

    public void GameOver() {
        gameOverPanel.SetActive(true);
        Time.timeScale = 0f;
        Cursor.lockState = CursorLockMode.None;
    }

    public void RestartGame() {
        Time.timeScale = 1f;
        SceneManager.LoadScene(SceneManager.GetActiveScene().name);
    }
}
🔧 User操作: シーンに空のGameObject「GameManager」を作成し GameManager.cs をアタッチ

動作確認チェックリスト

USER未
Unityエディタで再生し、以下の項目をすべて確認してください。
