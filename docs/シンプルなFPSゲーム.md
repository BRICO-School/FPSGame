# シンプルなFPSゲーム

## 概要
 - このプロジェクトは中学生向けのUnityの入門教材です
 - あまりにも高度な技術は使わず、Unityの基本的な仕組みのみで構築します

## Player
 - WASDで移動
 - 左クリックで射撃
 - Spaceでジャンプ
 - マウスで視点移動
 - 弾は30発（可変）
 - Rキーで弾をリロード


## Enemy
 - 自動的にプレイヤーを追跡
   - NAVMeshAgentを使用


## 共通仕様
 - キャラクターは敵、プレイヤー共に体力を持つ
 - 体力がなくなると消滅
   - プレイヤーが死んだらゲームオーバー画面に移行
 - 敵もプレイヤーも銃を持つ
   - 弾は30発（可変）
   - リロード中にリロードは出来ない
   - リロードは3秒かかる
